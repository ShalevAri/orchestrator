# Contributing to Orchestrator

Thank you for your interest in contributing!

## How to Contribute

### Adding a New Subagent

1. **Category** - Place your subagent in the most appropriate category
2. **Test your subagent** - Ensure it works with OpenCode
3. **Update required files** - When adding a new agent, update:
   - **Main README.md**: Add your agent to the appropriate category in alphabetical order
   - **AGENTS.md**: Add your agent to the appropriate category in alphabetical order
4. **Submit a PR** - Include a clear description of the subagent's purpose

### Pull Request Process

1. Fork the repository
1. Create a feature branch
1. Add your subagent following the template
1. Update all required locations
1. Submit a pull request with a clear description

### Quality Guidelines

- Subagents should be production-ready
- Provide practical examples
- Ensure compatibility with OpenCode
